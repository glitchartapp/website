Installation

1. From VS Code Marketplace

Search for "Binary Glitch Art Editor" in the VS Code extensions panel and click install, or use the direct link.

2. From Command Line

code --install-extension glitchartapp.binary-art-editor

3. Manual Installation

Download the .vsix file from GitHub releases and install using VS Code's "Install from VSIX" option.

Quick Start

# Right-click any file in VS Code Explorer

# Start creating glitch art!
